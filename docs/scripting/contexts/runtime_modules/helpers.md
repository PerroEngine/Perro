# Helpers

Helper macros:
- `sid!("name") -> ScriptMemberID`
- `var!("name") -> ScriptMemberID`
- `method!("name") -> ScriptMemberID`
- `func!("name") -> ScriptMemberID`
- `signal!("name") -> SignalID`
- `tag!("name") -> TagID`
- `tags!["a", "b"] -> Vec<TagID>`
- `variant!(value) -> Variant`
- `params![a, b, c] -> Vec<Variant>`
