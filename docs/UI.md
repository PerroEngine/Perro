# UI (.fur) Reference

This document describes the UI elements and attributes recognized by the FUR parser in `perro_core/src/nodes/ui/apply_fur.rs`. If something is not listed here, it is not parsed by the engine.

## Elements

Panel, Button, Text, Include.

## Base Attributes (All Elements)

These are parsed by `apply_base_attributes` and apply to all elements.

- `id` Element name/id used for scripting lookups (e.g. `get_element("main_panel")`).
- `pv` Pivot. Compound `x,y` or anchor tokens. Example: `pv=tl`, `pv=0.5,0.5`.
- `pv-x` Pivot X (float).
- `pv-y` Pivot Y (float).
- `tx` Translation X. Supports `%` (stored as relative).
- `ty` Translation Y. Supports `%`.
- `scl` Scale. Compound `x,y`. Supports `%`.
- `sz` Size. Compound `x,y`. Supports `%`.
- `w` Width. Supports `%` and `auto`.
- `h` Height. Supports `%` and `auto`.
- `rot` Rotation. Supports `%`.
- `z` Z index.
- `anchor` One of: `tl`, `t`, `tr`, `l`, `c`, `r`, `bl`, `b`, `br`.
- `visible` One of: `true`, `false`


## Panel

Attributes parsed in addition to base attributes:

- `bg` Background color (preset name or `#hex`, optional opacity suffix with `_NN`, e.g. `red-5_50`).
- `border` Border thickness (float).
- `border-c` Border color.
- `opacity` 0-1 or percent (`0.5`, `50%`). Applied to the panel and its children.
- `rounding` Corner radius. 1-4 values. Percent allowed (stored as `%`).
- `rounding-t`, `rounding-b`, `rounding-l`, `rounding-r` Directional overrides.
- `rounding-tl`, `rounding-tr`, `rounding-bl`, `rounding-br` Corner overrides.

Example:

```fur
[Panel w=15% h=15% anchor=c bg=lime-5 rounding=50% border=2 border-c=red-5]
  [Text fsz=25 align=c anchor=c]Center[/Text]
[/Panel]
```

## Button

Buttons are clickable panels. They accept all Panel attributes (`bg`, `border`, `rounding`, etc.), and their content comes from child elements (e.g. `[Text]`, nested panels/layouts).

Additional Button-only attributes:

- `hover-c` Background color when hovered. If omitted, the engine lightens `bg`.
- `pressed-c` Background color when pressed. If omitted, the engine darkens `bg`.
- `hover-border-c` Border color when hovered. If omitted, the engine lightens `border-c`.
- `pressed-border-c` Border color when pressed. If omitted, the engine darkens `border-c`.

## Text

Attributes parsed in addition to base attributes:

- `fsz` or `font-size` Font size (float).
- `font` Font file path or system font name.
- `align` Text flow alignment. `start|center|end` and aliases `s|c|e`, `left|right|top|bottom`, `l|r|t|b`.
- `align-h` Deprecated alias for horizontal alignment. Still supported.

Text content is taken from child text nodes and trimmed.

## Include

Special tag handled during conversion.

- `path` Path to another `.fur` file to include.
- `visible` Optional visibility gate for the included subtree.
